 📊 Task 2: Exploratory Data Analysis & Business Insights

📌 OBJECTIVE 
To analyze e-commerce sales data using Python, SQL, and Excel, and extract meaningful business insights.


🛠 TOOLS & TECHNOLOGIES USED 
- Python (Google Colab)
- Pandas, Matplotlib, Seaborn
- SQL
- Microsoft Excel (Pivot Tables & Dashboard)


📂 PROJECT STRUCTURE 
- EDA Notebook (Python)
- SQL Queries File
- Excel Dashboard File


🔍 KEY ANALYSIS PERFORMED 

1. Data Cleaning
- Handled missing values
- Converted date columns
- Filtered valid orders

2. KPI Analysis
- Total Revenue
- Total Orders
- Total Quantity Sold
- Average Order Value (AOV)

3. Category Analysis
- Revenue by product category
- Identified top-performing category

4. Monthly Trend Analysis
- Revenue trend over months
- Identified peak sales periods

5. Order Status Analysis
- Distribution of order statuses
- Identified cancellation patterns

6. State-wise Analysis
- Top states contributing to revenue


📊 Dashboard (Excel)
- KPI Cards
- Category-wise Revenue Chart
- Monthly Revenue Trend
- Filters for interactive analysis
  

💡 Key Insights
- “Set” category generates highest revenue
- Majority of orders are successfully delivered
- Certain months show peak sales trends
- Top states contribute significantly to overall revenue
